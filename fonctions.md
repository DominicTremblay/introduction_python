# Les fonctions

## Syntax de base
```python
def nom_fonction(parametres):
    # corps de la fonction
    return valeur  # optionnel

# Exemple simple
def action(nom="Mario", action="saute"):
    print(f"{nom} fait l'action de {action}!")

# Appel de la fonction
action()  # Mario fait l'action de saute!
action("Luigi", "courir")  # Luigi fait l'action de courir!
```
